# C# Examination System

A C# console-based examination system that simulates a simple exam environment where students can answer questions and receive automatic grading.

## Features
- Multiple question types:
  - True / False
  - Choose One
  - Choose All
- Automatic exam correction
- Practice exam mode (shows correct answers)
- Final exam mode (hides correct answers)
- Student notification when exam starts
- Question logging to a file
- Object-Oriented Design

## Technologies
- C#
- .NET Console Application
- OOP Concepts

## Concepts Used
- Inheritance
- Polymorphism
- Abstraction
- Interfaces
- Generics
- Delegates & Events
- Collections (List, Dictionary)

## How it works
1. The user selects the exam type (Practice or Final).
2. Questions are displayed.
3. The student enters answers.
4. The system evaluates answers automatically.
5. The final score is displayed.
